Welcome to Tetris!

Instructions:

  • As the blocks drop create a single, connected horizontal row
  • Use the 'left' and 'right' arrow keys to move the blocks left and right, respectively
  • Use the 'up' arrow key to rotate the block
  • When you're ready, press the 'Start/Pause' button to begin.
